BudgetApp (Java)


Info
  • The BudgetApp is a CRUD App made with Java OpenJDK 8 and the Swing library.

  • This App is made as a moduleassingment of module Designing and Programming (Ontwerpen en Programmeren) for my education HBO Software Development.

  • To prevent any copyright issues, the supermarketnames have been changed to supermarktnaam 01, 02, etc in this online version.

Creating and adding articles
  • The User can fill in the fields of: Weekbudget, Jaar, Weeknummer. The program will show the corresponding weekdate.

  • The User can choose a Supermarktnaam from a dropdown. The User can fill in an Artikelnaam, Prijs per stuk and choose Aantal from a dropdown. The program will calculate the Totaalprijs.

  • The User can check checkboxes to fill in if the article has been bought already and if it was a discount or not.

  • If the User presses the button "Add article" the program will check if all field are filled in correctly. If there are errors the program will inform the User by text but also with a red X at the field where the error occurred. If the field is valid, then a green V will appear at the corresponding field.

  • If all fields are filled in correctly the User is able to add the product to the list.

Deleting articles
  • The User can delete articles by first clicking on the row in the grocerylist and then click on the button delete.

Exporting grocerylist
  • When the User is satisfied with their list he or she can export the list to .txt format.

  • Per export an overview list of all supermarkets and a detailed list per supermarket will be made.

Regex IBAN searcher App (Java)

Info
  • A small simple program which searches a Dutch IBAN in a String or a txt file.

  • As the title of this project already shows, this program relies on the use of Regex patterns.

Usage - Searching a Dutch IBAN number in a String text
  • In this project a Dutch non-existent IBAN number is used with spaces as well as one without spaces.

  • The method searchIBAN_IN_Text takes the given String as an argument and searches for the IBAN number by a Regex pattern. If a match is found a message will be printed to inform the user. If no match is found another message will be printed to inform the user.

  • Possible exceptions in methods are caught using try/catch.
Usage - Searching a Dutch IBAN number in a file
  • In this project a Dutch IBAN number is searched in a file.

  • The method searchIBAN_IN_TextFile takes the relative path to the file order.txt as an argument.

  • The method searchIBAN_IN_TextFile first retrieve the absolute path to the file. Then by using a BufferedReader and a search Regex pattern it will open the file and look for a matching pattern of the Dutch IBAN number.

  • If a match is found it will inform the user with the Countrycode, the Country checkcode, the Bankcode and the Bankaccount numer.

  • Possible exceptions in methods are caught using try/catch.
